I created a fully admin panel to a boxing club. The aim of the software application was to provide a tool for handling affiliation of a customer to the club. The can handle payment, view on the client, basic operations on them, reports, statistics and more. A custom REST API was created for them and distributed in a web page. The database was on the cloud and fully secured for the operations.
